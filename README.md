# GS – Futuro do Trabalho 2025  
Sistema Inteligente de Recomendação de Carreira para o Futuro do Trabalho

## Descrição  
Este repositório apresenta o projeto desenvolvido como parte da atividade **Global Solution 2025** da :contentReference[oaicite:0]{index=0}, para a disciplina **Dynamic Programming / Engenharia de Software**.  
O objetivo é implementar um sistema de recomendação de trilhas de habilidades baseado em técnicas de Programação Dinâmica, Simulação de Monte Carlo, Algoritmos de Ordenação e Modelagem Probabilística — e aplicá-lo ao contexto real do Futuro do Trabalho.

## Como executar  
1. Acesse o notebook pelo link acima ou faça download do arquivo `.ipynb`.  
2. No Colab, vá em **Runtime → Run all** para executar todas as células sequencialmente.  
3. Verifique que as células de validação do grafo, DP, Monte Carlo, benchmark e recomendação rodaram sem erros.  
4. No final, salve os resultados ou exporte gráficos conforme necessário.

## Requisitos e Dependências  
- Python 3.x  
- Bibliotecas: `numpy`, `matplotlib`, `statistics` (já incluídas no Colab), `docx` (se for exportar relatório via script)  
- Runtime recomendado: GPU/TPU não necessário; execução pode ser feita em CPU comum.
